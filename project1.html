Create your Personal Webpage

About the project:

In this project you will create your personal webpage. This project will provide you an opportunity to practice the basic concepts of HTML. You will not use any Webpage editors such as Dreamweaver or FrontPage for this project. You may use text editor to create this page. Personal webpage may contain information about yourself such as your education background, your favorite sports or movies, your hobbies, your interests etc.

Points: 100

Due Date:

Folder Structure and Naming Conventions:

****Your assignment will not be graded if you do not follow the following folder structure and naming conventions.

  • Your project will include one base folder using the following naming convention:
    • IMD131_proj1_firstnameLastname (example: IMD131_proj1_johnDoe)
  • The base folder must have one “images” folder inside it. All your images must reside in the “ images”folder.
  • The base folder must also have “index.html” web page at the root of the base folder.

Project Requirements:

  1. Try to be creative.
  2. You webpage must look Professional. (10 points)
  3. You must use this link to check the markup validity of your webpage in XHTML. (15 points)
  4. No grammar or spelling errors. (5 points)
  5. Your webpage must have a Title (<title> tag) (5 points)
  6. Your page will start with the following document type declaration and language tag (Copy and paste it into your webpage) (10 points)

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" " http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" lang="en" xml:lang="en">

Must contain at least one of each of the following:

  1. External link with title attribute(All external links must open in a new window) (5 points)
  2. Anchor link (10 points)
  3. Image (Make sure you do not violate any copyright laws while using images and every image must have ALT tags.) (15 points)
  4. List elements(10 points)
  5. Two styles (CSS) (15 points).
