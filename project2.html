Create a website for your School

About the project:

This project is an opportunity for you to present the practical and conceptual skills that you have learned during this class and integrate them into a well built Website. You have plans to start your own Graduate School and you have decided to create a Website for your school.

  • The Website should have at least 5 web pages. Additional pages are allowed.
  • I will upload your work on to a web server so your classmates can see it.
  • Your project will not be graded if you do not follow the following folder structure and naming conventions.

Points: 100

Due Date:

Folder Structure and Naming Conventions:

****Your assignment will not be graded if you do not follow the following folder structure and naming conventions.

  • Your project will include one base folder using the following naming convention:
    • IMD131_proj2_firstnameLastname (example: IMD131_proj2_johnDoe)
  • The base folder must have one "images" folder inside it. All your images must reside in the “images ”folder.
  • The base folder must also have “index.html” (which is your “About Us” page) web page at the root of the base folder.

Project Requirements:

  1. Try to be creative.
  2. You can use Dreamweaver or any other HTML editor for this project.
  3. The Index / About Us page should describe how you want your school to be.
  4. The site should exhibit proper use of background, font, and colors.
  5. Website must look Professional.
  6. No grammar or spelling errors.
  7. Use only one external style sheet for the whole website.
  8. Your page will start with the following document type declaration and language tag (Copy and paste it into your webpage)

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" " http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml " lang="en" xml:lang="en">

HTML Requirements:

  1. The Website should have at least 5 web pages which must include an Index page, Reference page and Contact us page.
  2. Make sure your XHTML and CSS validate.
  3. All links should be properly linked and working.
  4. Each page must have an appropriate Title
  5. The Contact Us page should have a Form with at least:
    1. 2 Text fields
    2. Two radio buttons
    3. One select menu
    4. One check box
    5. One Submit button and
    6. One Reset button to reset all the fields in the form.
  6. Every page should have an image and all images must have alt tags. No broken images please. (Make sure you do not violate any copyright laws while using images.)
  7. There should be one references page which should have links to the sites from which you may have used any images or content.
  8. The Website should have at least one div and one table.

JavaScript Requirements:

  1. Use “onmouseover” and “onmouseout” events.
  2. On the ‘Contact Us’ page, when you click the Submit button, an alert window should pop up with the values in the two text fields.
  3. A date and time Script on “About us” page.
